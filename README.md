# Feihong's Hugo Quickstart

## Installation

    asdf plugin add hugo
    asdf list-all hugo
    asdf install hugo latest
    asdf global hugo latest
    hugo version

## Commands

Create a new site

    hugo new site quickstart

Build the static HTML files

    make build

Start the development server

    make server
